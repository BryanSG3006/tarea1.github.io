¿Qué es SEO?

Pagina principal

SEO (Search Engine Optimization) se puede entender como la práctica de optimizar el contenido y la estructura de un pagina web para mejorar su visibilidad en los motores de búsqueda como Google. Esto implica una serie de técnicas y estrategias que se implementan en el código HTML de la pagina web

Consejos para hacer una web optimizada para SEO

Algunos consejos que seran importantes a tomar en cuenta para lograr posicionar de mejor manera nuestra pagina web serian los siguientes:

Debemos usar titulos que sean claros con el tema, para que sean atractivas y se logre entender de mejor manera lo que deseamos transmitir con nuestra pagina web, con esto la gente que busca la pagina lograra entender de que se trata dicha pagina

Podemos incluir palabras claves, estas palabras deberan ser relacionadas con lo que el usuario va a buscar en el motor de busqueda, debera estar implementado en los textos , titulos o descripciones

Otra forma de hacerlo es usar encabezados más ordenados , usando las diferentes etiquetas que tiene HTMl como lo son H1,H2,H3 , usando el H1 como titulo principal y sucesivamente según el enfasis que deseamos darle al título

Otro aspecto a tener en cuenta es la velocidad de la pagina web, debemos asegurarnos que la velocidad de carga sea rapido , podemos hacerlo optimizando imagenes, usando un buen hosting y reduciendo tamaño del codigo

Hacer un correcto modo de adaptado dependiendo el dispositivo movil que usemos, ya sea celular, tablet o computadora

Crear contenido de calidad, escribiendo datos e informacion importante y unica con temás que sean de relevancia para la audiencia esperada de la pagina

Meta Etiquetas

Algunas de las principales Meta Etiquetas

Meta Etiqueta de Descripcion

Esta meta etiqueta tiene el proposito de mostrar un resumen del contenido de nuestra pagina web

Su principal uso es al momento de mostrar tu pagina web en el motor de busqueda, ya que este muestra dicha descripcion debajod el título de la pagina, Debe ser atractiva y contener palabras claves que ayudaran a posicionar dicho sitio web

Meta Etiqueta de Palabras Claves

Esta meta etiqueta tiene el proposito de listas las palabras claves relevantes para el contenido de la página

En un principio era utilizado por los motores de busqueda para lograr comprender de que va a tratar la página

Meta Etiqueta de Robots

El proposito es indicar a los mtores de búsqueda como indexar la página web

Valores comunes que incluyen serian , index, follow, noindex entre otros

Meta Etiqueta ViewPort

Esta meta tiene el proposito de controlar cómo se muestra la página en dispositivos moviles

Ayuda a que la página sea responsiva , un valor común es content="width=device-width, initial-scale=1", este ajusta el ancho de la página al ancho del dispositivo y establece un zoom inicial adecuado

Meta Etiqueta de Autor

El propsito es brindar una informacion sobre el autor del contenido de la página

Es útil para asi dar credito al autor y puede ser utilizado por algunos motores de búsqueda y navegadores para mostrar esta informacion

Relación de Meta Etiquetas y SEO

Meta Etiqueta De Descripcion

Aunque no se vea afectado directamente tener una descripcion bien redactada ouede mejorar la tasa de clics y visibilidad de nuestro sitio web

Meta Etiquetas de Palabras Clave

En la actualidad, la mayoría de los motores de búsqueda, incluyendo Google, ya no utilizan la meta etiqueta de palabras clave para el ranking

Meta Etiquetas de Robots

Esta etiqueta controla cómo los motores de búsqueda rastrean e indexan tu página. Puedes indicar si quieres que una página sea indexada o no, y si los enlaces en la página deben ser seguidos o no

MetaE Etiqueta de ViewPort

Afecta la usabilidad y la experiencia del usuario en dispositivos móviles. Google utiliza la usabilidad móvil como un factor de ranking, por lo que una buena configuración del viewport puede mejorar el SEO.

Meta Etiqueta de Autor

Directamente no impacta el ranking de búsqueda, pero puede agregar valor al proporcionar información sobre el autor, lo cual puede ser útil para el reconocimiento de marca y la credibilidad
